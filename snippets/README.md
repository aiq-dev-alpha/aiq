# UI Snippets Library

Organized, copy-pastable UI components and flows for Flutter, SwiftUI, Jetpack Compose, React, React Native, and Angular with Ionic.

## Structure

```
snippets/
├── apps/                    # Full app templates (e.g., ecommerce)
├── u_flow/                  # User flows (checkout, return, onboarding)
├── state/                   # State management patterns
├── data/                    # Data layer (API, models, repositories)
├── navigation/              # Navigation/routing patterns
└── screens/                 # Individual screens (login, profile, etc.)
```

## Frameworks

- **Flutter** (`/flutter`)
- **SwiftUI** (`/swiftui`)
- **Jetpack Compose** (`/compose`)
- **React** (`/react`)
- **React Native** (`/react-native`)
- **Angular + Ionic** (`/angular-ionic`)

## Usage

Each snippet is production-ready with:
- Modern architecture patterns
- Responsive design
- Accessibility support
- Type safety
- Error handling
